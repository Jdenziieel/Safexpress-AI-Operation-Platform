# Screenshots Folder

Store your test screenshots here organized by test:

## Folder Structure
```
screenshots/
├── test1-simple-search/
│   ├── 01-before.png
│   ├── 02-supervisor-terminal.png
│   ├── 03-gmail-agent-terminal.png
│   └── 04-ui-results.png
│
└── test2-multi-step/
    ├── 01-before.png
    ├── 02-supervisor-terminal-plan.png
    ├── 03-gmail-agent-search.png
    ├── 04-pending-action.png
    ├── 05-gmail-agent-forward.png
    └── 06-ui-success.png
```

## Screenshot Naming Convention
- Use descriptive names
- Number them in sequence (01, 02, 03...)
- Include what's being shown (terminal, ui, before, after, etc.)

## Tools
- Windows Snipping Tool: `Win + Shift + S`
- Or use Print Screen: `PrtScn` key
